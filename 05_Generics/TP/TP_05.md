## Deskripsi Program
Program ini mendefinisikan sebuah fungsi hitung untuk menghitung jumlah karakter dalam sebuah teks berdasarkan dua mode, yaitu dengan menggunakan perulangan for...of untuk mengecek setiap karakternya. Jika mode "semua" dipilih, fungsi akan menghitung seluruh karakter termasuk spasi (menghasilkan angka 11 untuk teks "Bar bar bar"), sedangkan jika mode "huruf" dipilih, fungsi akan menggunakan perintah continue untuk melewati spasi dan hanya menghitung sisa karakternya saja (menghasilkan angka 9). Hasil dari proses tersebut dikembalikan menggunakan return dan dicetak ke layar melalui console.log(), namun pemanggilan fungsi pada baris paling akhir tidak menampilkan apa pun karena meskipun mesin tetap melakukan proses perhitungan, hasilnya tidak diperintahkan untuk dicetak atau disimpan ke dalam sebuah variabel.
